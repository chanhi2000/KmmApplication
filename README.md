# KmmApplication

__Kotlin Multiplatform__ Project using Jetpack Compose and SwiftUI. Currently running on

- Android (Jetpack Compose)
- iOS (SwiftUI)


## Requirements

 - Android Studio:  `4.2~`
     - Gradle: `6.8.2`
     - Android Plugin: `4.2.0`
     - Kotlin: `1.4.31`
 - XCode: `12.0~`
